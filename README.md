# SQL Portfolio — Hospital Database Project

A hands-on SQL project covering 10 core SQL concepts
using a hospital management database.

## Topics Covered
- SELECT, WHERE, ORDER BY, LIMIT
- Aggregate Functions & GROUP BY
- JOINs (INNER, LEFT, Multi-table)
- Subqueries & EXISTS
- CTEs, Views, Window Functions
- UNION, INTERSECT, EXCEPT
- Conditional Aggregation & CASE

## Files
- hospital_queries.sql — Full schema + practice queries
- SQL_Practice_Guide.html — Interactive practice guide
