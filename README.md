# 🏥 Hospital Database — SQL Portfolio Project

A hands-on SQL project built from scratch using MySQL Workbench,
covering 10 core SQL concepts through a hospital management database.

## 📁 Files
- `hospital_queries.sql` — Full schema, sample data & practice queries
- `my_sql_queries.sql` — Extended queries across multiple databases
- `SQL_Practice_Guide.html` — Interactive bilingual practice guide (EN + Tamil)

## 🧠 Topics Covered
1. SELECT, DISTINCT, Aliases
2. WHERE, Filtering & Operators
3. ORDER BY & LIMIT
4. Aggregate Functions
5. GROUP BY & HAVING
6. JOINs (INNER, LEFT, Multi-table)
7. Subqueries & EXISTS
8. CTEs & Views
9. Window Functions (ROW_NUMBER, RANK, LEAD, LAG)
10. UNION, INTERSECT, EXCEPT & Conditional Aggregation

## 🛠️ Tools Used
- MySQL 8.0
- MySQL Workbench
